# ChemReactionHelper
## Установка и запуск
### Установка
Чтобы собрать Docker образ, выполните следующую команду в корневой директории проекта:

```bash
docker-compose build
```
### Запуск приложения

Запустите приложение с помощью следующей команды:

```bash
docker-compose up
```
